Text To Phonetic

Text To Phonetic is one of my more recent projects. It is a simple tool to convert English to phonetic pronunciation. A few months ago I wanted to know how to pronounce a word, So I looked for a pronunciation guide. The commonly held idea that anything can be found on the internet is false. Every text-to-pronunciation converter and pronunctiation guide I could find used IPA symbols. I can't understand IPA, so I created this.
Visit Text To Phonetic

AutoCookieClicker

AutoCookieClicker is a enhancement/cheat script for Cookie Clicker. It enables Auto Click, automatic claiming of golden cookies, automatic buying of properties and upgrades as they become available, and billions of cookies with the click of a button. AutoCookieClicker is activated using a bookmarklet.

Speck

Speck is a WebGL-powered 3D atomic structure renderer. Speck was originally created by Rye Terrell. Thanks to Rye for making his project open source, I was able to make this improved version which has the added the capability to search a database of structure files by compound name to avoid the inconvenience of having to search for and enter xyz file data.
