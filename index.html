Text To Phonetic

Text To Phonetic is for those who have trouble understanding the International Phonetic Alphabet. It uses a custom-built dictionary to convert (nearly) any English word or sentence to the phonetically-spelled equivalent.

RND Trees

RND Trees is a mod for the Minecraft-like voxel game Minetest. It adds naturally-growing trees with highly customizable materials and growth parameters. You can even watch these trees grow.
Source on Github

Speck

Speck is a WebGL-powered 3D atomic structure renderer. Speck is not, properly speaking, my project. It is an open-source project created by Rye Terrell, who has not maintained it since 2015. My version remedies a major inconvenience of the original, that of finding properly formatted molecular models to import. An integrated search box allows searching of a database of structure files by compound name, as well pulling information about the selected compound from Wikipedia.
Visit Speck
Source on Github

Bertrand's Drawception ANBT

Bertrand's Drawception ANBT was a mod of an existing enhancement script for Drawception. The original version was made by Grom PE, was generously made open source. In addition to Grom PE's excellent new canvas and game and forum improvements, my version featured an improved site menu with dynamic colors based on game panels, as well as a complete dark mode, a menu option to browse all games, desktop notifications for new comments, and more. Sadly, this project has been discontinued as I no longer have time to keep up with updates to the Drawception website.
