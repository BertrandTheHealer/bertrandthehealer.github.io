Text To Phonetic

Text To Phonetic is one of my more recent projects. It is a simple tool to convert English to phonetic pronunciation. A few months ago I wanted to know how to pronounce a word, So I looked for a pronunciation guide. The commonly held idea that anything can be found on the internet is false. Every text-to-pronunciation converter and pronunctiation guide I could find used IPA symbols. I can't understand IPA, so I created this.
Visit Text To Phonetic

Speck

Speck is a WebGL-powered 3D atomic structure renderer. Speck was originally created by Rye Terrell. Thanks to Rye for making his project open source, I was able to make this improved version which has the added the capability to search a database of structure files by compound name to avoid the inconvenience of having to search for and enter xyz file data, as well as showing basic information about the selected compound.
Visit Speck

Bertrand's Drawception ANBT

Bertrand's Drawception ANBT is an enhancement script for Drawception. It is an improved version of Drawception ANBT by Grom PE, which he has generously made open source. In addition to Grom PE's excellent new canvas and game and forum improvements, my version features an improved site menu with dynamic colors based on game panels, as well as a complete dark mode, a menu option to browse all games, desktop notifications for new comments, and more.
